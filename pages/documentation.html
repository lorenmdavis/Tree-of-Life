Tree Of Life

Each section of content is contained using the 'branch' class.

Columns are wrapped in rows using the 'stem' class.

Columns are wrapped in containers using the 'leaf' class.

There can be up to 12 columns per row.

Content is one column at small screen, and can expand at larger sizes.
